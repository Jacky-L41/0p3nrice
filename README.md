# Accounts
Integrated accounts: 
student (no password)
demo (no password)

### After clone
Add your mongodb connection url to models/mongodb_url.js

```
echo "const url = ''; module.exports = url;" > models/mongodb_url.js
```

### Installing
```
npm install
```
### Running
```
npm start
```
### Testing
Go to http://localhost:8099
